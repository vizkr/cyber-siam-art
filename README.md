## 🎨 Style

All images are original AI-generated artwork created for the Cyber Siam music project.

- **Dark / Spiritual series** — cinematic dark photography aesthetic, temple interiors, neon city, wire and light
- **Klong Life series** — charcoal drawing with selective colour splashes, canal scenes, graphic novel style
